    Le symbole scatter est une partie intégrante presque de tous les machines à sous modernes avec les niveaux bonus.

    Il y a seulement un type de symboles qui ne peuvent pas être remplacé par le Symbole Wild, ce sont les symboles scatter. Le symbole scatter (ou dispersion) a reçu son nom selon sa nomination.

    Habituellement, seulement dans les cas, quand la combinaison des mêmes symboles forment une ligne de paiement, le joueur reçoit un prix en argent. Les symboles scatter ne doivent pas être nécessairement dans une même ligne pour former une combinaison gagnante. Quand 3 ou plus de symboles de ce genre apparaissent sur les rouleaux, ils déclenchent une action prescrite (par exemple, des tours bonus, des tours supplémentaires gratuits, le multiplicateur, etc.).

    Les symboles scatter ne sont pas toujours banals. En outre, les développeurs de logiciels sans arrêt produisent des nouveaux et originaux modèles de machines à sous avec les autres fonctions inhabituelles de ces symboles.
